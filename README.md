# Pat Cummins IPL Performance Dashboard

## Overview
This project is an interactive IPL analytics dashboard built using Tableau.  
The dashboard analyzes Pat Cummins' IPL batting and bowling performance across multiple seasons.

## Tools Used
- Tableau
- Microsoft Excel

## KPIs Included
- Total Wickets
- Economy Rate
- Bowling Average
- Total Runs
- Batting Strike Rate
- Matches Played

## Charts Included
- Wickets by Season
- Economy by Season
- Bowling Average by Season
- Runs by Season
- Batting Strike Rate by Season
- Boundary Analysis

## Features
- Interactive season filter
- Highlight actions
- SRH-themed dashboard design
- Professional sports analytics layout

## Key Insights
- Pat Cummins recorded his highest wickets in IPL 2024.

- He maintained a strong all-round contribution across seasons.
